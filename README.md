# Essay Abridger

Essay Abridger is a simple AI-powered web application built with **Python**, **Streamlit**, **LangChain**, and **Groq**. It converts long essays into concise abridgements while preserving the main ideas and important details.

## Features

* AI-powered essay summarization
* Fast responses using Groq LLMs
* Simple and user-friendly interface


## Technologies Used

* Python
* Streamlit
* LangChain
* Groq API
